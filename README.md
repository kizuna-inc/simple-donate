## Tech Stats

### FrontEnd

- Prototype

  - JS: `Vue.js` + `typescript` + `vite.js`
  - CSS: `Bootstrap 5`

- Production

  - JS: `Next.js` + `typescript`
  - CSS: `tailwind.css`

### BackEnd

- `express.js`, `typescript`
- ORM
  - `Prisma`
- DB

  - `PostgreSQL`

- w/ Streaming Overlay
  - using `HTML`, `CSS`, `JS (Vanilla + Socket.io)` // >:D
